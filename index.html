Explorando CSS

Cor em CSS

A propriedade de cor em CSS permite definir a cor do texto e dos elementos. Pode-se utilizar valores como nomes de cores (ex.: red, blue), códigos hexadecimais (#RRGGBB), RGB (rgb(R, G, B)), entre outros.

Centralização em CSS

Para centralizar elementos em CSS, pode-se usar propriedades como text-align: center; para centralizar texto dentro de elementos, e margin: 0 auto; para centralizar blocos de conteúdo horizontalmente dentro de seu contêiner.

Tamanho em CSS

A propriedade de tamanho em CSS, geralmente usada em conjunto com width e height, permite definir as dimensões de elementos como caixas de texto, imagens e outros blocos. Pode-se usar unidades como px (pixels), em, rem, %, entre outros.

Tags HTML: <p>, <h> e <img>

A tag <p> é usada para criar parágrafos de texto.

As tags <h1> até <h6> são usadas para definir cabeçalhos, onde <h1> é o mais importante e <h6> é o menos importante.

A tag <img> é usada para inserir imagens em uma página HTML, utilizando o atributo src para indicar o caminho da imagem.
